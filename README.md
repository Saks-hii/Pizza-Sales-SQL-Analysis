# Pizza Sales Analysis using SQL

## Project Overview

This project analyzes a pizza sales dataset using SQL to answer real-world business questions and generate insights related to sales performance, customer behavior, and revenue trends.

---

## Skills Demonstrated

- SQL
- Aggregate Functions
- GROUP BY
- ORDER BY
- JOINS
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Business Analysis

---

## Business Questions Solved

✔ Total Revenue Generated

✔ Total Number of Orders

✔ Most Common Pizza Size

✔ Highest Priced Pizza

✔ Top 5 Most Ordered Pizza Types

✔ Category-wise Sales Analysis

✔ Average Number of Pizzas Ordered Per Day

✔ Percentage Revenue Contribution by Pizza Type

✔ Cumulative Revenue Analysis

✔ Hourly Distribution of Orders
